# it-helpdesk-lab

A simulated lab that demonstrates ticketing workflows, SLA management, and IT troubleshooting.

## Tools Used
osTicket (SupportSystem)

## What's Covered
- Five realistic helpdesk tickets.
- Network, security, and software scenarios.
- Detailed descriptions of each tickets and their resolutions.
